# 申请流程
- 企业需填写申请表，链接：https://ausn3mvz6ps0.larksuite.com/share/base/form/shrus9q7mP0K7tywsAhVSNFRMDd
- 提交后，平台将对申请进行审核，审核通过后会发放试用账号及相关API-key，每个客户仅限申请一次试用。
# 账号及API-key管理
- 试用账号有效期为**14天**，账号下所有API-key与账号有效期绑定。到期后，AI服务将自动终止，但查询用量服务仍可继续使用。
- API-key的创建及用量查询，请访问：https://datav.seendoing.com
- **试用期自首次调用AI服务时生效**。为避免浪费试用时长，请在调用前确保内部沟通和需求规划已完成。
# 使用建议与准备
- 在开始试用前，请确保：
  - **IT部门与业务部门**就试用场景进行了充分沟通和需求规划；
  - 相关数据授权和准备工作已完成，以保证服务顺利开展。
# 使用规范与支持
- 在试用期间，请严格遵守以下规范：
  - **合规使用数据**，确保数据隐私与安全；
  - **禁止滥用API**或获取未经授权的数据；
  - 不得用于非法用途。
- 若在试用期间发现不合规行为，知行奇点有权**收回账号权限**并提前终止试用资格。

如有任何疑问或需要技术支持，请随时联系我们的团队。
